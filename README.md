# התפצלות הספירות

אתר אינטראקטיבי המציג את עץ החיים הקבלי והתפצלות הספירות.

## קישור לאתר

https://alfasin.github.io/sphirot/

## הוראות הפעלה

1. לחץ על ספירה כדי לראות את התפצלותה לעשר ספירות נוספות
2. המשך ללחוץ כדי להעמיק ברמות נוספות (עד אינסוף)

## התפצלות

כל ספירה מכילה בתוכה את כל עשר הספירות:
- 10 ספירות → 100 ספירות → 1000 ספירות וכן הלאה
